# Fitness Tracker
Description: An app that allows you to log your workouts and log into to access previous logs.

## Event Handler Functions
- `openDropdown(event)` - utilizes `createDropdown` function to open a dropdown menu consisted of `drop` objects inside the `mainMenu` array of objects
- `showForm(type)` - function that applies form template to Log In and Sign Up objects when clickled


## Helper Functions
- `makeElement(el)` - function that creates an element
- `createDropdown(a, item)` - function that adds `drop` objects to a div named dropDown
